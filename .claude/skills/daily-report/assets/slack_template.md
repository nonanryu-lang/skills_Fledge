# Slack 日報テンプレート

以下の形式で日報を生成する。`{...}` の部分をユーザーの情報で埋める。

---

```
📅 *日報 {YYYY/MM/DD（曜日）}*

━━━━━━━━━━━━━━━━━━
✅ *今日の作業内容*
• {作業内容1}
• {作業内容2}
• {作業内容3}

📈 *進捗・達成事項*
• {達成事項1}
• {達成事項2}

📌 *明日の予定*
• {予定1}
• {予定2}
━━━━━━━━━━━━━━━━━━
```

---

## 記載ルール

- 各セクションは箇条書き（`•`）で列挙する
- 項目が1つしかない場合もそのまま箇条書きで記載する
- 作業・予定が特にない場合は「特になし」と記載する
- 日付の曜日: 月・火・水・木・金・土・日
- コードブロック（` ``` `）で囲むことでSlackでの見た目を整える
